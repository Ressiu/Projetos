CRIE O SITE DOS SEUS SONHOS

Olá, eu sou o Ressiu, Web Developer

Olá! Meu nome é Ressiu e sou um desenvolvedor front-end iniciante com conhecimento básico e mediano em HTML, CSS e JavaScript. Estou constantemente aprendendo e aprimorando minhas habilidades para criar experiências web incríveis.

O site atual em que você está navegando é um projeto pessoal que criei para mostrar uma espécie de vitrine de produtos. Esses produtos são na verdade pacotes de serviços de desenvolvimento, e o site foi projetado para exibir esses serviços de uma forma única. Em vez de mostrar fotos dos produtos, optei por usar ícones e textos que descrevem detalhadamente cada serviço oferecido. O resultado é uma vitrine intuitiva, repleta de interatividade e personalidade.

Estou muito animado com este projeto, pois me permite aplicar e experimentar minhas habilidades de desenvolvimento web de uma maneira prática e criativa. Espero que você goste de explorar o site tanto quanto eu gostei de criá-lo!
